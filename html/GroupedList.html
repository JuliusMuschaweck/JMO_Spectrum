Generate spectra

CIE_Illuminant returns spectra for standard CIE illuminants

CIE_Illuminant_D Computes CIE standard illuminant D (daylight) for a desired color temperature

GaussSpectrum creates a Gaussian spectrum, normalized to peak = 1.0

PlanckSpectrum computes Planck blackbody spectra

Add, modify, transform spectra

AddSpectra adds two spectra without weights.

AddWeightedSpectra adds several spectra with weights>

IntegrateSpectrum computes integral of spectrum over wavelength, with an optional weighting function

MatchAdditiveMix computes weights of three XYZ tristimuli to match a target XYZ tristimulus, with an optional fixed contribution

ResampleSpectrum replaces the wavelength data points, e.g. to ensure a spectrum is based on 360:830 nm in 1 nm steps

Evaluate colorimetric properties

CCT computes the correlated color temperature of a spectrum

CCT_from_xy computes the correlated color temperature from a CIE 1931 xy color point

CIE1931_XYZ computes CIE 1931 tristimulus values X, Y, Z and x, y, z color coordinates for a spectrum

CIE_Lab Computes CIELAB L*, a*, b* values from XYZ tristimulus values

CIE_Luv Computes CIELUV L*, u*, v* values from XYZ tristimulus values

CIE_upvp Computes CIE 1960/1964 u, v, u', v' color coordinates from CIE 1931 x, y

CIEDE2000_Lab Computes the CIEDE2000 color difference between two CIELAB L*, a*, b* stimuli

CIEDE2000_XYZ Computes the CIEDE2000 color difference between two XYZ stimuli.

ComputeSpectrumColorimetry For a given spectrum, computes a large selection of colorimetric properties

CRI The CRI class provides what is needed to compute Color Rendering Indices%% LED model

LDomPurity computes dominant wavelength and purity

Utility functions

CauchyFromAbbe computes A0 and A1 coefficients of the Cauchy dispersion model from reference index and Abbe number

CIE1931_Data returns struct with CIE 1931 color matching functions, monochromatic border and Planck locus

CODATA2018 Returns a struct with CODATA 2018 constants relevant to illumination, and some more

EvalSpectrum evaluates the function modeled by a spectrum

FindRoot1D finds root of scalar function of one real variable, without derivatives.

FindRootND finds multidimensional root of function in N dimensions, without derivatives

IsOctave determines if program is running under GNU Octave

IsSpectrum checks if a variable is a valid spectrum

JMOSpectrumVersion returns the version of this library

LinInterpol from tabulated function yy(xx), compute linearly interpolated values at xq query points

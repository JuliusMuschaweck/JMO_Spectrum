ResampleSpectrum

Navigate to:   Home   |   Alphabetic list   |   Grouped list Source code: ResampleSpectrum.m

Assign a new wavelength vector to a spectrum, interpolating the old values.

Syntax

function rv = ResampleSpectrum(spectrum, lam_new)

Input Arguments

  • spectrum: scalar, valid spectrum (see SpectrumSanityCheck)
  • lam_new: 1-D vector of double, valid wavelength array (positive, stricly ascending)

Output Arguments

  • rv: A copy of the old spectrum with all fields, but lam is replaced and val is linearly interpolated. For lam_new values outside the old interval, val(i) == 0

Usage Example

function ExampleResampleSpectrum()
    s_old = MakeSpectrum([400 500], [0 100]);
    s_new = ResampleSpectrum(s_old, [450 451 452])
end
